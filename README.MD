# vue-module-lab

Repositorio de laboratorio para módulos UI en Vue, utilidades compartidas y assets compilados.

## Propósito general
- Centralizar componentes Vue reutilizables.
- Probar flujos de módulos por dominio (producciones, actividades, lotes, etc.).
- Mantener separación entre UI, lógica de dominio y persistencia.

## Estructura principal
- `resources/assets/js/` → código fuente principal de frontend.
- `resources/assets/sass/` → estilos fuente.
- `public/` → artefactos compilados.
- `docs/` → documentación funcional y operativa.
